\# AGENTS.md



\## Project

This repo is a high-converting landing page for AgentFlow by Gen I Labs.



\## Goal

Build a premium one-page landing page that converts real estate agents and small real estate teams into booked calls or install inquiries.



\## Product rules

\- AgentFlow is the hero offer.

\- AgentFlow is specifically for real estate.

\- Gen I Labs is the parent brand behind the offer.

\- Sell outcomes, not generic AI hype.

\- Do not imply native WhatsApp automation/sending.

\- Do not imply direct website lead capture integrations.

\- Do not imply full SaaS.

\- Do not imply custom integrations unless explicitly requested.



\## Conversion rules

\- Every section should support one clear next step.

\- The page should feel premium, direct, trustworthy, and easy to act on.

\- Prioritize clarity, trust, and motion toward booking.

\- Use real-estate-specific pain points and outcomes.



\## Design rules

\- Dark premium SaaS aesthetic

\- Emerald green accents

\- Cinematic, minimal, modern

\- Strong spacing and hierarchy

\- Mobile responsive

\- Subtle interactivity only where it improves conversion



\## Technical rules

\- Use Next.js App Router

\- Use TypeScript

\- Use Tailwind CSS

\- Keep components reusable and clean

\- No backend

\- No auth

\- No database

\- No CMS

\- No Convex



\## Working rules

\- Read BRAND\_BRIEF.md, OFFER.md, and COPY\_NOTES.md before making changes.

\- Explain major file structure changes before making them.

\- Use placeholder content only where necessary.
